> Disclaimer: This project is provided for educational and testing purposes only. Use at your own risk. Author is not responsible for any damage, data loss, or legal issues that may result from applying this code to any software or system.

### How To Use
Clone the repo, and open the directory:
```bash
git clone https://github.com/ExclMark/crossres.git
cd crossres
```

Run the code:
```bash
python3 patch.py
```
